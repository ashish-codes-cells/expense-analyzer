# Expense Analyser

This is a begginer-friendly data analysis project built using Python.

## Project Overview
This project analyzes personal expense data and provides useful insights about spending patterns.

## Tools Used
- Python
- Pandas
- Matplotlib

## Features
- Total expense calculation
- Category-wise expense analysis
- Data Visualization using bar chart
- Basic insights and spending suggestions

## Key Insights
- Highest expense category identified
- Lowest expense category identified
- Suggestions for optimizing expenses

## Files
- expenses.csv - dataset
- expense_tracer.ipynb - main project notebook
